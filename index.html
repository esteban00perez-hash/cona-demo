      Como se calcula el rating
      

        

          Cada jugador arranca con una base de 6.0 puntos. Durante la mejenga se suman o restan puntos segun las acciones registradas:
        

        

          
Gol+0.8

          
Asistencia+0.5

          
Equipo ganador+0.5

          
Salvada clave+0.35

          
Atajada (portero)+0.2

          
Accion defensiva+0.1

          
Tiro al arco+0.1

          
Palo / Tiro afuera bueno+0.05

          
Gol recibido (portero)-0.2

          
Gol recibido (campo)-0.05

          
Tiro afuera malo-0.2

          
Autogol-1.0

        

        

          Despues de calcular los puntos raw, se normalizan con el Factor Cona para que los ratings queden entre 6.5 y 9.0. Si un jugador se destaca claramente del resto, puede llegar a 10.0.
